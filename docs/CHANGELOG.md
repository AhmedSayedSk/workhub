# Changelog

All notable changes to WorkHub will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2025-03-18

### Added

- Project management with multi-tier hierarchy (Organizations, Systems, Projects, Features, Tasks, Subtasks)
- Kanban board with drag-and-drop, priorities, task types, and confetti celebrations
- Live time tracking with persistent timer across page navigation
- Calendar with month/week/day/list views, drag-and-drop events, and category color coding
- Financial management with milestone, monthly, and fixed-price payment models
- Media library with drag-and-drop upload, folder hierarchy, and image compression
- Project vault with passkey-protected sensitive data storage
- AI assistant powered by Google Gemini with task breakdowns, time estimates, and web search
- Notification system with deadline alerts, payment reminders, timer reminders, and break reminders
- Dark mode with full theme support
- Docker support for development
- Firebase security rules and storage rules
- MCP server for Claude Code time tracking integration
